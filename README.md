# WebHub — 个人网页工作台

把常用网页变成 APP，支持自定义外观和自动化操作。

## 功能

| 模块 | 说明 |
|------|------|
| 🏠 **多工作区** | 2–8 个独立工作区，每个有独立 WebView，切换秒显 |
| 🔗 **多链接** | 每个工作区可配多个链接，底部菜单或标题栏切换 |
| 🖥️ **桌面模式** | 每个链接独立开关，电脑版网页竖屏全宽可见 |
| 🎨 **名称图标切换** | 7 套预设，设置中随时换桌面图标和应用名 |
| ⚙️ **自定义操作** | 隐藏/点击/修改/脚本 四种操作，支持 CSS 选择器 |
| ⏱️ **定时刷新** | 每个工作区独立刷新间隔 |
| 🌙 **夜间模式** | App UI + 网页 CSS 双重暗色 |
| 🔍 **内容搜索** | 遍历所有已打开 WebView 搜索页面正文 |
| 🔎 **元素检查** | 点击页面任意元素查看选择器，方便配置操作 |
| 📤 **导入导出** | JSON 格式一键备份恢复全部配置 |

## 操作类型

| 操作 | 说明 | 延迟 |
|------|------|------|
| 隐藏 | 移除页面元素 | 无 |
| 点击 | 自动点击元素 | 可设置 |
| 修改 | 替换元素文字 | 无 |
| 脚本 | 注入自定义 JS | 可设置 |

生效范围：仅此链接 / 相似域名 / 当前工作区 / 所有工作区。

## 名称图标预设

设置 → 应用设置 → 应用名称和图标，7 套可选：

| # | 名称 | 图标 |
|---|------|------|
| 1 | WebHub | 默认蓝 |
| 2 | WebHub | 自定义 |
| 3 | LanHub | 网络 |
| 4 | ECM | CRM |
| 5 | Gming | 自定义 |
| 6 | Pili | 自定义 |
| 7 | PILI | 自定义 |

切换后返回桌面，图标和名称几秒内刷新。

## 桌面模式

每个链接卡片内的开关。原理：桌面 UA + viewport `width=980` + `initial-scale` 按屏幕宽度计算，页面全宽缩放可见，支持双指放大操作。
