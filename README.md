# WebHub - 个人网页工作台

把常用网页变成 APP，支持自定义外观和自动化操作。

---

## ✨ 功能特性

### 核心功能

- 📱 **多标签管理** - 支持 2-5 个选项卡，每个选项卡可配置多个链接
- 🎨 **自定义操作** - 隐藏/点击/修改页面元素
- ⏱️ **延迟点击** - 点击操作可设置延迟执行
- 🔍 **元素检查器** - 查看页面元素标识，方便配置操作
- 🔄 **智能刷新** - 短按缓存刷新，长按强制刷新
- 🔙 **智能返回** - 金山文档弹窗自动关闭

### 设置功能

- ⚙️ **灵活配置** - 选项卡、链接、操作三级管理
- 📤 **导出/导入** - JSON 格式备份恢复
- 🧹 **缓存管理** - 一键清除缓存
- 🔌 **金山文档优化** - 可开关的弹窗关闭功能

## 🚀 快速开始

### 安装

1. 下载 `WebHub.apk`
2. 传输到手机
3. 安装（允许未知来源）

### 配置

1. 点击右上角 **⋮** 菜单
2. 选择 **⚙️ 设置**
3. 配置选项卡、链接、操作
4. 点击 **保存设置**

### 使用

- **短按**选项卡：切换视图
- **长按**选项卡：显示子链接菜单
- **短按**刷新：缓存刷新
- **长按**刷新：强制刷新
- **返回键**：金山文档弹窗自动关闭

## 📋 功能说明

### 自定义操作

| 操作 | 说明 | 延迟 |
|------|------|------|
| **隐藏** | 隐藏页面元素 | 无 |
| **点击** | 自动点击元素 | 可设置 |
| **修改** | 修改元素文字 | 无 |

### 元素选择器

| 类型 | 格式 | 示例 |
|------|------|------|
| Class | `.类名` | `.top-banner` |
| ID | `#id名` | `#header` |

### 元素检查器

1. 点击 **⋮** 菜单
2. 选择 **🔍 查找元素**
3. 点击页面元素
4. 弹窗显示元素信息
5. 点击 **复制选择器**

---

## 📦 导入导出

### 导出

1. 进入设置
2. 点击 **📤 导出设置**
3. 保存为 JSON 文件
4. 可分享给他人

### 导入

1. 进入设置
2. 点击 **📥 导入设置**
3. 选择 JSON 文件
4. 自动恢复配置
